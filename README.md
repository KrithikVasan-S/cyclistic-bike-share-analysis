# Cyclistic Bike-Share Case Study

##  Overview
This project analyzes bike-share data to understand how casual riders and annual members use the service differently. The goal is to provide insights that help increase annual memberships and improve business profitability.

---

##  Business Problem
The company aims to convert casual riders into annual members. This analysis identifies key behavioral differences between the two user groups.

---

##  Tools Used
- R
- tidyverse
- lubridate
- ggplot2

---

##  Key Insights
- Casual riders use the service more during weekends, indicating leisure usage  
- Members show higher activity during weekdays, suggesting commuting behavior  
- Casual riders take longer rides compared to members  
- Members have peak usage during typical working hours  

---

##  Recommendations
- Introduce weekend promotions targeting casual riders  
- Offer trial memberships to encourage conversion  
- Focus marketing on high-traffic and tourist locations  
- Highlight cost benefits of membership for frequent users  

---


##  Project Files
- `BikeShare_Project.Rmd` – Full analysis code  
- `BikeShare_Project.html` – Final report ( Download and open in a browser )  


---

##  Conclusion
The analysis highlights clear behavioral differences between casual riders and members. Targeted marketing strategies can help convert casual users into annual members, improving overall profitability.
